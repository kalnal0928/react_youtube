# YouTube Downloader v3.0.0

YouTube 동영상을 쉽고 빠르게 다운로드할 수 있는 데스크톱 애플리케이션입니다.

## ✨ 주요 기능

- 🎬 **최대 10개 URL 동시 다운로드**
- 📊 **실시간 진행률 표시** - 부드러운 애니메이션
- 🎯 **다양한 품질 옵션** - 최고 품질부터 480p까지
- 🎵 **MP3 추출 지원** (FFmpeg 필요)
- 🔄 **동적 큐 시스템** - 다운로드 중에도 URL 추가 가능
- 📝 **실시간 로그** - 다운로드 상태 확인
- 🎨 **현대적인 UI** - Material-UI 기반

## 🚀 사용 방법

### 다운로드 및 설치

1. [Releases](../../releases) 페이지에서 최신 버전 다운로드
2. `YouTube-Downloader-v3.0.0.zip` 압축 해제
3. `YouTube Downloader.exe` 실행

**설치 불필요!** 압축 해제 후 바로 사용 가능합니다.

### 기본 사용법

1. **URL 입력**
   - YouTube URL을 입력창에 붙여넣기
   - 최대 10개까지 입력 가능
   - Ctrl+V로 자동 추가

2. **품질 선택**
   - 최고 품질 (병합) - FFmpeg 필요
   - 최고 품질 (단일 파일) - 권장
   - 720p HD
   - 480p
   - 음성만 (MP3)

3. **다운로드 경로 설정**
   - 기본: `내 문서/Downloads/YouTube`
   - 찾아보기 버튼으로 변경 가능

4. **다운로드 시작**
   - "다운로드 시작" 버튼 클릭
   - 실시간 진행률 확인
   - 다운로드 중에도 URL 추가 가능

## 📋 시스템 요구사항

- **운영체제**: Windows 10/11 (64bit)
- **메모리**: 4GB RAM 이상 권장
- **저장공간**: 100MB + 다운로드 파일 용량
- **FFmpeg**: 고급 기능 사용 시 필요 (선택사항)

## 🔧 FFmpeg 설치 (선택사항)

FFmpeg는 다음 기능에 필요합니다:
- 최고 품질 병합
- MP3 추출

### 설치 방법

1. 앱 내 "?" 버튼 클릭
2. FFmpeg 설치 가이드 확인
3. 설치 후 "새로고침" 버튼으로 확인

또는 [FFmpeg 공식 사이트](https://ffmpeg.org/download.html)에서 다운로드

## 💡 팁

- **빠른 추가**: URL 복사 후 Ctrl+V만 누르면 자동 추가
- **동적 큐**: 다운로드 중에도 새 URL 추가 가능
- **로그 확인**: "로그 보기" 버튼으로 상세 정보 확인
- **폴더 열기**: "폴더 열기" 버튼으로 다운로드 폴더 바로 열기

## 🐛 문제 해결

### 다운로드가 시작되지 않아요
- yt-dlp 업데이트 버튼 클릭
- 인터넷 연결 확인
- URL이 올바른지 확인

### 진행률이 표시되지 않아요
- 로그 보기로 다운로드 상태 확인
- 예상 시간 기반 진행률이 표시됩니다

### FFmpeg 관련 오류
- FFmpeg 설치 확인
- 환경 변수 PATH 설정 확인
- 앱 재시작
