# Binary Files

이 폴더에는 다음 파일들을 배치해야 합니다:

## 필수 파일
- `yt-dlp.exe` - YouTube 다운로드 엔진
  - 다운로드: https://github.com/yt-dlp/yt-dlp/releases/latest
  - Windows용 실행 파일을 다운로드하여 이 폴더에 배치

## 선택적 파일
- `ffmpeg.exe` - 비디오/오디오 처리 (고급 기능용)
  - 다운로드: https://ffmpeg.org/download.html
  - Windows 빌드를 다운로드하여 bin 폴더에 배치

## 주의사항
- 이 파일들은 Git에 포함되지 않으므로 수동으로 다운로드해야 합니다.
- 배포 시에는 이 파일들이 포함되어야 합니다.
- 라이선스 조건을 확인하고 준수해야 합니다.
